# 🛒 쇼핑 리스트 앱

아이템 추가·삭제·체크 기능이 있는 심플한 쇼핑 리스트 웹 앱입니다.  
별도 서버 없이 `index.html`을 브라우저에서 바로 열어 사용할 수 있습니다.

## 기능

- **아이템 추가** — 입력 후 `+` 버튼 또는 `Enter`
- **체크** — 구매완료 표시 (취소선 + 초록 체크)
- **삭제** — 개별 항목 삭제
- **필터** — 전체 / 미구매 / 구매완료 탭 전환
- **일괄 삭제** — 완료 항목 한 번에 정리
- **데이터 유지** — `localStorage`에 자동 저장

## 사용 방법

```
index.html 파일을 브라우저로 열기
```

## 테스트 실행

```bash
npm install
npm test
```

Playwright 기반 자동 테스트 22개 케이스를 실행합니다.
